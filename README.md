# Computation Worksheets

These are worksheets that I've created in order to teach programming.

There are no answer keys, as there are no objective answers when solve computation problems.  Unless it's to trace a program, in which case you can just run the code.

The worksheets provided were taught with Python as the main language to problem solve with.  If there comes a time where I am teaching in a language other than Python, I will add a column to the table below specifying language.

## Available Worksheets

| Worksheet | Level | Description |
| -------- | ------- | ------- |
| AI Prompt Practice  | Beginner | Computation tasks provided with expectation to provide prompts to AI agent. |
| Control Flow | Beginner | Python code snippets provided with expectation to provide output |
